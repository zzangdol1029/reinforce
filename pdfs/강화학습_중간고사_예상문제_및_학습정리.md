## 2. Ch.02 Bandit Problem

### 학습해야 할 핵심 내용
- **Multi-armed Bandit** 문제 정의
- **Action Value** Q(a) = E[R | A=a]
- **Sample-average estimation**과 **Incremental update**
  $$ Q_n = Q_{n-1} + \frac{1}{n}(R_n - Q_{n-1}) $$
- **Exploitation(활용) vs Exploration(탐색)**
- **ε-greedy 정책**: 확률 ε로 random, 1-ε로 greedy
- **Stationary vs Non-stationary** 문제
- 비정상 문제에서의 **고정 step size α** 사용:
  $$ Q_n = Q_{n-1} + \alpha (R_n - Q_{n-1}) $$
- 가중치가 지수적으로 감소: 최신 보상에 더 큰 가중치

### 예상 문제
**[Q2-1]** Multi-armed bandit 문제에서 action value Q(a)의 정의를 쓰고, 표본평균에 의한 추정식과 그것을 점진적(incremental)으로 표현한 식을 유도하라.

**[Q2-2]** ε-greedy 정책을 설명하고, ε=0과 ε=1일 때 각각 어떤 정책이 되는지 설명하라.
- ε=0 → 완전한 greedy(exploitation only)
- ε=1 → 완전한 random(exploration only)

**[Q2-3]** Stationary와 Non-stationary bandit 문제의 차이를 설명하고, 비정상 문제에서 step size를 1/n 대신 고정값 α로 사용하는 이유를 설명하라.
- 답: 1/n은 모든 보상에 동일 가중치(과거=최신). 고정 α는 최신 보상에 더 큰 가중치를 부여하므로 환경 변화 추종이 가능.

**[Q2-4] (코딩/유도)** 슬롯머신을 3회 플레이하여 보상이 [1, 0, 5] 였을 때 incremental 식을 사용하여 Q1, Q2, Q3을 단계별로 계산하라. (Q0=0 가정)
- Q1 = 0 + (1/1)(1-0) = 1
- Q2 = 1 + (1/2)(0-1) = 0.5
- Q3 = 0.5 + (1/3)(5-0.5) = 2.0

## 3. Ch.03 Markov Decision Process (MDP) ★★★ 가장 중요

### 학습해야 할 핵심 내용
- **Markov Property**: P(S_{t+1} | S_0,...,S_t) = P(S_{t+1} | S_t)
- **MDP 5튜플**: (𝒮, 𝒜, T, r, γ)
- **Episodic task vs Continuing task**
- **Return**:
  - Episodic: G_t = R_{t+1} + R_{t+2} + ... + R_T
  - Continuing: G_t = R_{t+1} + γR_{t+2} + γ²R_{t+3} + ...
- **할인율 γ**의 역할 (0 ≤ γ ≤ 1, continuing task에서 발산 방지)
- **State-value function**: $v_\pi(s) = \mathbb{E}_\pi[G_t | S_t=s]$
- **Action-value function**: $q_\pi(s,a) = \mathbb{E}_\pi[G_t | S_t=s, A_t=a]$
- **Bellman Equation (state-value)**:
  $$ v_\pi(s) = \sum_{a,s'} \pi(a|s) p(s'|s,a) [r(s,a,s') + \gamma v_\pi(s')] $$
- **Bellman Equation (action-value)**:
  $$ q_\pi(s,a) = \sum_{s'} p(s'|s,a) [r(s,a,s') + \gamma \sum_{a'} \pi(a'|s') q_\pi(s',a')] $$
- **Bellman Optimality Equation**:
  $$ v_*(s) = \max_a \sum_{s'} p(s'|s,a)[r(s,a,s') + \gamma v_*(s')] $$
- **Optimal policy**: $\mu_*(s) = \arg\max_a q_*(s,a)$

### 예상 문제 (★ 출제확률 매우 높음)
**[Q3-1]** Markov Property의 정의를 수식으로 쓰고, 의미를 한 줄로 설명하라.
- "현재 상태가 주어지면 미래는 과거와 독립이다(Future ⊥ Past | Present)"

**[Q3-2]** Episodic task와 Continuing task에 대한 Return G_t의 정의를 각각 쓰고, continuing task에서 할인율 γ가 필요한 이유를 설명하라.
- 답: γ<1 이어야 무한 합이 유한한 값으로 수렴

**[Q3-3]** State-value $v_\pi(s)$와 Action-value $q_\pi(s,a)$를 정의하고, 둘 사이의 관계식을 유도하라.
$$ v_\pi(s) = \sum_a \pi(a|s) q_\pi(s,a) $$

**[Q3-4] ★ 핵심** State-value에 대한 Bellman 방정식을 다음 두 단계를 거쳐 유도하라.
- (1) $G_t = R_{t+1} + \gamma G_{t+1}$ 관계 사용
- (2) 기댓값의 선형성 $\mathbb{E}[X+Y]=\mathbb{E}[X]+\mathbb{E}[Y]$ 사용

**[Q3-5] (Quiz Q1 변형)** 다음 1x5 Grid World에 대한 transition diagram과 r(s,a,s')을 모두 표로 작성하라.
- State: {L1, L2, L3, L4, L5}, 사과(+1), 폭탄(-2), 빈칸(0), 벽(0)

**[Q3-6] (Quiz Q2 변형)** 1x2 Grid World (γ=0.9)에서 결정적 정책 μ(s)에 대한 state value $v_\mu(L1), v_\mu(L2)$를 연립방정식으로 계산하라.
- L1↔L2 (right=+1, left=0인 경우 등)에 대해 직접 풀이 연습

**[Q3-7] (Quiz Q3 변형)** 4-Grid World 문제(L1=-1, L4=+1, γ=0.9)에서 균등(stochastic) 정책에 대한 Bellman 연립방정식을 4개 미지수에 대해 작성하고 풀이하라.

**[Q3-8] (Quiz Q4 변형)** 같은 4-Grid World 문제에 대해 Bellman optimality equation을 4개 상태에 대해 유도하라.

## 4. Ch.04 Dynamic Programming ★★★

### 학습해야 할 핵심 내용
- **DP의 본질**: 큰 문제를 작은 부분 문제로 분해, 결과 저장
- **차원의 저주(curse of dimensionality)** 문제점
- **Iterative Bellman equation**: $V_0 \to V_1 \to ... \to V_k \to V_{k+1} \to ...$
- **Bootstrapping**: 다음 상태의 추정치로부터 현재 상태 추정치 갱신

#### Policy Evaluation (정책 평가)
$$ V_{k+1}(s) = \sum_{a,s'} \pi(a|s) p(s'|s,a)[r(s,a,s') + \gamma V_k(s')] $$

#### Policy Improvement (정책 개선) - Greedy
$$ \mu'(s) = \arg\max_a \sum_{s'} p(s'|s,a)[r(s,a,s') + \gamma v_\pi(s')] $$

#### Policy Iteration vs Value Iteration
| 항목 | Policy Iteration | Value Iteration |
|------|-----------------|-----------------|
| 절차 | 평가(수렴까지) → 개선 반복 | 평가 1회 + 개선 1회 통합 반복 |
| Update 식 | π → 평가 → 개선 → π' | $V_{k+1}(s) = \max_a \sum_{s'} p[r+γV_k]$ |
| 수렴속도 | 평가 비용 큼 | 평가 1회씩이라 빠름 |
| 정책 필요 | 평가 시 π 필요 | 평가 시 π 불필요 (max 사용) |

### 예상 문제 (★ 출제확률 매우 높음)
**[Q4-1]** Bellman 방정식을 직접 푸는 것(연립방정식)의 어려움을 설명하고, 이를 DP로 푸는 핵심 아이디어인 **iterative Bellman equation**과 **bootstrapping** 개념을 설명하라.

**[Q4-2]** Iterative Policy Evaluation 알고리즘의 의사코드(pseudo code)를 작성하라. 종료 조건도 포함하라.

**[Q4-3] ★** Policy Iteration과 Value Iteration의 차이를 표로 비교하라. 특히 update 식과 수렴 속도, 정책 필요 여부의 관점에서.

**[Q4-4]** Policy Improvement Theorem의 statement를 쓰고, 이 정리가 보장하는 바가 무엇인지 설명하라.
- "모든 s에 대해 $q_\pi(s, \mu'(s)) \ge v_\pi(s)$ 이면 $v_{\pi'}(s) \ge v_\pi(s)$ 이다."

**[Q4-5] (Quiz 변형)** 2x2 또는 3x4 Grid World에서 다음 중 하나를 한 step 수동 계산하라.
- (a) Policy Evaluation: 균등 정책에서 V_0 → V_1
- (b) Value Iteration: V_0 → V_1 (max 적용)
- (c) Greedy improvement: 주어진 V로 새 μ(s) 도출

**[Q4-6]** DP 방법의 두 가지 한계점을 쓰시오.
- (1) 차원의 저주 (state/action 수 증가시 계산량 폭증)
- (2) 환경 모델 p(s'|s,a)와 r(s,a,s')을 알아야 함 (model-based)

## 5. Ch.05 Monte Carlo Method ★★

### 학습해야 할 핵심 내용
- **Monte Carlo 정의**: 무작위 샘플링 반복으로 수치적 결과 도출
- **분포 모델 vs 샘플 모델**
- **MC Prediction (Policy Evaluation)**:
  - 에피소드를 끝까지 진행하여 G 계산
  - $V_n(s) = V_{n-1}(s) + \frac{1}{n}(G - V_{n-1}(s))$
  - **Learning** (DP는 computing)
- **MC Control (Policy Improvement)**: Q(s,a) 추정 + ε-greedy
- **Action-value 사용 이유**: 모델 없이 정책 개선하려면 V가 아닌 Q 필요
- **ε-greedy in MC**: exploration 보장
- **Exponential Moving Average**: 1/n 대신 고정 α 사용 → 최신 데이터 가중

### DP vs MC 비교 (출제확률 매우 높음)
| 항목 | DP | MC |
|------|-----|-----|
| 환경 모델 | 필요(O) | 불필요(X) |
| 차원의 저주 | 있음(O) | 없음(X) |
| Episode | 일회성/지속성 모두 가능 | 일회성만(O), 지속성(X) |
| Markov Property | 필요(O) | 불필요(X) |
| 학습 방식 | Bootstrapping (computing) | Sampling (learning) |
| 평가 시점 | 매 step | 에피소드 종료 후 |

### 예상 문제
**[Q5-1]** Monte Carlo 방법의 개념을 설명하고, 원주율 π 또는 주사위 눈의 합 기댓값 계산을 예로 들어 동작 원리를 설명하라.

**[Q5-2] ★** DP와 MC의 차이를 6가지 항목(환경 모델, 차원의 저주, episode 종류, Markov property, 학습 방식, 평가 시점)으로 비교하는 표를 작성하라.

**[Q5-3]** MC Prediction에서 V(s)를 incremental하게 갱신하는 식을 유도하라.
$$ V_n(s) = V_{n-1}(s) + \frac{1}{n}[G^{(n)} - V_{n-1}(s)] $$

**[Q5-4]** MC Control에서 V(s) 대신 Q(s,a)를 추정해야 하는 이유를 환경 모델 관점에서 설명하라.
- 답: V만으로는 모델 p(s'|s,a)가 있어야 정책 개선이 가능하지만, Q를 직접 추정하면 모델 없이도 $\mu(s) = \arg\max_a Q(s,a)$로 개선 가능.

**[Q5-5]** 1/n 갱신 대신 고정 α를 사용하는 EMA(Exponential Moving Average) 방식의 장점과 가중치 변화를 설명하라.

**[Q5-6] (Quiz 변형)** 5x5 Grid World에서 MC Control을 적용했을 때 ε와 α의 변화에 따른 학습 결과 차이를 설명하라.

## 6. Ch.06 Temporal Difference Method ★★★

### 학습해야 할 핵심 내용
- **TD = MC + DP** : sampling(MC) + bootstrapping(DP)
- **TD vs MC vs DP**:
  - DP: bootstrap O, sampling X
  - MC: bootstrap X, sampling O
  - TD: bootstrap O, sampling O
- **TD(0) Update**:
  $$ V(S_t) \leftarrow V(S_t) + \alpha[R_{t+1} + \gamma V(S_{t+1}) - V(S_t)] $$
- **TD Target**: $R_{t+1} + \gamma V(S_{t+1})$
- **TD Error**: $\delta_t = R_{t+1} + \gamma V(S_{t+1}) - V(S_t)$
- TD 장점: 매 step 갱신, 일회성+지속성 모두 가능, 분산 작음

#### SARSA (On-policy TD Control)
- 5튜플 (S_t, A_t, R_{t+1}, S_{t+1}, A_{t+1})로 갱신
- $$ Q(S_t,A_t) \leftarrow Q(S_t,A_t) + \alpha[R_{t+1} + \gamma Q(S_{t+1},A_{t+1}) - Q(S_t,A_t)] $$
- 정책 개선: ε-greedy

#### On-policy vs Off-policy
- **On-policy**: 행동 정책 = 대상 정책 (SARSA)
- **Off-policy**: 행동 정책 ≠ 대상 정책 (Q-Learning, Off-policy SARSA)
- Off-policy는 **importance sampling** 필요 (단, Q-Learning은 max라서 불필요)

#### Importance Sampling
$$ \mathbb{E}_\pi[x] \approx \frac{1}{n}\sum \rho(x^{(i)}) x^{(i)}, \quad \rho(x) = \frac{\pi(x)}{b(x)} $$
- π와 b가 비슷할수록 분산 작음

#### Q-Learning (Off-policy TD Control)
$$ Q(S_t,A_t) \leftarrow Q(S_t,A_t) + \alpha[R_{t+1} + \gamma \max_a Q(S_{t+1},a) - Q(S_t,A_t)] $$
- Bellman **optimality** equation의 sampling 버전
- max를 사용하므로 importance sampling 불필요 → 안정적

### SARSA vs Q-Learning 비교
| 항목 | SARSA | Q-Learning |
|------|--------|------------|
| 기반 식 | Bellman equation | Bellman optimality equation |
| Target | $R + \gamma Q(S', A')$ | $R + \gamma \max_a Q(S', a)$ |
| 정책 종류 | On-policy | Off-policy |
| Importance Sampling | 불필요(on-policy) / 필요(off-policy SARSA) | 불필요 |
| 학습 안정성 | 보수적 | 더 적극적 최적화 |

### 예상 문제 (★ 출제확률 매우 높음)
**[Q6-1] ★** TD 방법이 DP, MC와 어떻게 다른지 표로 비교하라(bootstrapping, sampling, episode 종류, 갱신 시점, 분산 등).

**[Q6-2] ★** TD(0) 알고리즘의 V(s) 갱신식을 쓰고, **TD target**과 **TD error**를 표시하라. MC의 갱신식과의 차이도 설명하라.

**[Q6-3]** SARSA 알고리즘 의사코드를 작성하라. 이름이 'S-A-R-S-A'인 이유를 설명하라.

**[Q6-4] ★** On-policy와 Off-policy의 차이를 정의하고, 각각의 예를 들어라.
- On: SARSA
- Off: Q-Learning, Off-policy SARSA

**[Q6-5]** Importance Sampling의 수식을 쓰고, off-policy SARSA에서 보정 비율 ρ가 어디에 곱해지는지 설명하라.

**[Q6-6] ★ 핵심** SARSA와 Q-Learning의 차이를 다음 관점에서 비교하라.
- (1) 사용하는 Bellman 방정식 (equation vs optimality equation)
- (2) Target에서 다음 행동 선택 방식 (샘플링 vs max)
- (3) On/Off-policy
- (4) Importance sampling 필요 여부
- (5) 학습 안정성

**[Q6-7]** Q-Learning이 off-policy임에도 importance sampling이 불필요한 이유를 설명하라.
- 답: Target에서 $\max_a Q(S',a)$를 사용하므로, 행동 정책 b로부터 샘플링한 A_{t+1}이 사용되지 않기 때문.

## 7. 종합 비교 (꼭 외워야 하는 표)

### Policy Evaluation 갱신식 비교
| 방법 | 식 |
|------|-----|
| DP | $V_{k+1}(s) = \sum_{a,s'} \pi(a\|s) p(s'\|s,a)[r + \gamma V_k(s')]$ |
| MC | $V_n(s) = V_{n-1}(s) + \frac{1}{n}[G - V_{n-1}(s)]$ |
| TD | $V(S_t) \leftarrow V(S_t) + \alpha[R + \gamma V(S_{t+1}) - V(S_t)]$ |

### Policy Control 알고리즘 비교
| 방법 | 정책 평가 | 정책 개선 | On/Off-policy | 모델 필요 |
|------|----------|----------|--------------|----------|
| Policy Iteration (DP) | 반복 평가 | greedy | - | O |
| Value Iteration (DP) | 1-step + max | greedy | - | O |
| MC Control | 에피소드 종료 후 Q 갱신 | ε-greedy | On | X |
| SARSA | TD Q 갱신 | ε-greedy | On | X |
| Off-policy SARSA | TD + IS | greedy(target) / ε-greedy(behavior) | Off | X |
| Q-Learning | TD + max Q 갱신 | greedy | Off | X |
