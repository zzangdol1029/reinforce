# V-value와 Q-value — 가치함수·정책평가 정리

> **관련 문서:** [시간차(TD) 방법 — SARSA — Q-Learning](./06-Temporal_Difference_Method.md) (TD, SARSA, Q-Learning 흐름)

---

## 질문에서의 “p-value”에 대해

통계학에서 **p-value**는 가설 검정의 **유의확률**을 뜻합니다. 강화학습 맥락에서는 오타로 **상태 가치 \(V\)** 또는 정책 기호 **\(\pi\)** (Pi)를 뜻했을 가능성이 있습니다. 아래에서는 **가치함수**와 **정책평가**, 그리고 **Q-value**와 **V-value**의 차이를 정리합니다.

---

## 1. 가치함수와 정책평가의 개념

### 가치함수 (Value Function)

에이전트가 현재 상태 \(S_t\)에서 출발하여 목표에 도달할 때까지 앞으로 얻을 수 있는 **기대 수익(Return)** 의 추정치입니다.

### 정책평가 (Policy Evaluation)

에이전트가 현재 설정된 행동 지침(**정책 \(\pi\)**)을 따랐을 때, 그 정책이 얼마나 좋은지를 확인하기 위해 **가치함수를 계산하고 갱신하는 과정**입니다. 평가가 끝나면 이를 바탕으로 더 나은 행동을 찾는 **정책 갱신(Policy Update)** 을 수행합니다.

---

## 2. V-value (State-value) vs Q-value (Action-value)

가치함수는 **무엇을 기준으로 평가하느냐**에 따라 두 가지로 나뉩니다.

### V-value (상태 가치함수, State-Value Function)

| 항목 | 내용 |
|------|------|
| **정의** | 특정 **상태(State)** 자체가 얼마나 가치 있는지를 평가합니다. |
| **수식** | \(V^\pi(s)\) (정책 \(\pi\) 하에서의 상태 가치) |
| **핵심 질문** | “지금 이 **위치(상태)** 에 있는 것만으로 앞으로 얼마나 보상을 받을 수 있을까?” |

### Q-value (행동 가치함수, Action-Value Function)

| 항목 | 내용 |
|------|------|
| **정의** | 특정 상태에서 **특정 행동(Action)** 을 취했을 때의 가치를 평가합니다. SARSA·Q-Learning에서 정책을 결정하고 갱신하는 **직접적인 대상**이 됩니다. |
| **수식** | \(Q^\pi(s,a)\) |
| **핵심 질문** | “지금 이 상태에서 **‘오른쪽으로 이동’** 같은 **행동**을 하면 앞으로 얼마나 보상을 받을 수 있을까?” |

---

## 3. 핵심 요약 비교

| 구분 | V-value (상태 가치함수) | Q-value (행동 가치함수) |
|------|-------------------------|-------------------------|
| **평가 대상** | 현재 상태 \(s\) | 현재 상태 \(s\) + 취할 행동 \(a\) |
| **수학적 표현** | \(V(s)\) | \(Q(s,a)\) |
| **활용 알고리즘** | DP(동적 계획법) 기반 평가, TD로 \(V\) 추정 등 | SARSA, Q-Learning 등 |
| **특징** | 행동을 고르기 위해 **환경 모델**(다음 상태 예측)이 추가로 필요한 설정이 많음 | 상태와 행동이 묶여 있어 **모델 없이(Model-free)** 행동을 선택·갱신하기 쉬움 (예: \(\max_a Q(s,a)\)) |
